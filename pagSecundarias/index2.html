1. Java: Año de creación: 1995 Creado por: James Gosling, en Sun Microsystems. Descripción: Java es un lenguaje de programación orientado a objetos, diseñado para ser portátil, seguro y eficiente. Su principio "escribe una vez, ejecuta en cualquier parte" permite que los programas escritos en Java puedan ejecutarse en cualquier dispositivo con una máquina virtual Java (JVM). Es muy utilizado en desarrollo de aplicaciones empresariales, Android y sistemas de backend. 2. Python:
Año de creación: 1991 Creado por: Guido van Rossum. Descripción: Python es un lenguaje de programación de alto nivel, conocido por su sintaxis simple y clara, lo que facilita su aprendizaje y uso. Es ampliamente utilizado en desarrollo web, ciencia de datos, automatización, inteligencia artificial y más. Su comunidad y gran cantidad de bibliotecas lo hacen extremadamente versátil. 3. JavaScript (JS):
Año de creación: 1995 Creado por: Brendan Eich, en Netscape. Descripción: JavaScript es un lenguaje de programación que se ejecuta en el navegador y es fundamental para el desarrollo web. Permite crear contenido interactivo, desde animaciones hasta la validación de formularios en tiempo real. Con el tiempo, ha evolucionado y ahora también se usa en el lado del servidor gracias a entornos como Node.js. 4. CSS (Cascading Style Sheets):
Año de creación: 1996 Creado por: Håkon Wium Lie. Descripción: CSS es un lenguaje de hojas de estilo utilizado para describir la apariencia y el diseño de las páginas web. Permite definir cómo se mostrarán los elementos HTML, incluyendo colores, fuentes, márgenes y disposición. Junto con HTML y JavaScript, es uno de los pilares del diseño web moderno.
